# What is a Computer?

A **computer** is an electronic device that accepts data as input, processes it according to a set of instructions, and produces meaningful output.

---

## Basic Functions of a Computer

A computer performs four main operations:

1. **Input** – Receiving data  
2. **Processing** – Performing calculations  
3. **Storage** – Saving information  
4. **Output** – Displaying results  

---

## Example

For example:

- When you type numbers in a calculator app
- The computer processes the calculation
- And displays the result instantly.

---

## Components of a Computer

### Input Devices

Devices used to enter data into the computer.

Examples:

- Keyboard
- Mouse
- Scanner

### Processing Unit

The **CPU (Central Processing Unit)** processes instructions and performs calculations.

### Output Devices

Devices used to display results.

Examples:

- Monitor
- Printer
- Speakers
