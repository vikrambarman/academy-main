# Introduction to MS Word

## MS Word Kya Hai?

Microsoft Word ek **word processor** software hai jo documents
banane ke liye use hota hai.

## Features

- Text formatting (Bold, Italic, Underline)
- Tables banana
- Images insert karna
- Page setup

## Practical Exercise

1. MS Word open karo
2. Naya document banao (Ctrl+N)
3. Apna naam type karo
4. **Bold** karo (Ctrl+B)
5. Save karo (Ctrl+S)
